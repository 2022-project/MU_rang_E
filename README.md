#### 메인 기능

    1️⃣ 음악 추천 :  사용자의 표정을 5초간 분석 후, 해당 감정에 맞는 음악을 추천해줍니다.

    2️⃣ 감정 달력 : 분석 후, 사용자의 감정이 유저 달력에 색상으로 저장됩니다. (가장 높은 수치의 감정 2가지를 기준으로 색상이 정해집니다)
